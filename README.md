# 解放碑 (Liberation Stele)

**高温量子相干光热计算薄膜 · 全光推理与科学发现平台**

---

## 一句话

一张在 242°C 下运行、用太阳光驱动、单次乘加能耗仅 6 fJ 的有机共晶薄膜——既能做大规模矩阵推理，也能做可编程量子物理实验。

---

## 它是什么

**一张 DiSubPc·C70 有机共晶薄膜，四层行星级架构，一个开源承诺。**

| 维度 | 说明 |
|------|------|
| **物理核心** | 242°C 高温量子相干窗口，利用激子-极化激元的集体量子行为完成光域点积运算 |
| **能效** | 单次乘加 6 fJ（纯光学）/ 17 fJ（系统级），比电子计算低约 **6 万倍** |
| **功能** | 可热重写的全光自注意力加速器，同时可作为可编程量子模拟器探究新物理 |
| **架构** | 自然光终端 → 邻居互助光路 → 赤道光环 → 散射备援 → 双面太阳帆全球星座 |

```bash

cd photothermal-liberation-stele
```
