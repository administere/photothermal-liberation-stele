## 物理原理

1. **Key 存储**：K 矩阵以折射率调制形式存储在 DiSubPc·C70 共晶薄膜（200 nm）中，薄膜在 242°C 的精密 PID 恒温下工作
2. **Query 注入**：Q 向量编码为光子脉冲序列，以 10 ns 间隔串行注入
3. **光域乘法**：利用量子相干拍频窗口（~1.2 ps）完成光子-激子相干乘加
4. **探测器读出**：乘法结果经 InGaAs 探测器阵列读出

## 操作数修正说明

本仿真使用物理正确的 MAC 计数：

$$\text{MACs} = \text{seq\_len}^2 \times d$$

这是 Q·K^T 的实际点积次数：每个 $(i,j)$ 位置的元素都需要一次 $d$ 维点积。  
**而非**常见的错误公式 $\text{seq\_len} \times d \times \text{size}$。
